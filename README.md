# Versionador

Projeto em Perl [Este projeto tem por objetivo, criar um sistema minimalista de controle de versão como o Subversion(SVN)]

## Instalação
[Copie o binario (versionador.pl) para um diretório em algum computador com linux e Perl]

## Uso
Edite o arquivo versionador.pl e inclua os arquivos que serão versionado com o caminho relativo.

[Execute: ./versionador.pl] sempre antes e depois de alguma modificação em seu código
